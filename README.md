# UCU Partnership Growth Strategy Dashboard

An interactive data dashboard analyzing University Credit Union's growth opportunities across its 14+ university partner schools.

**Live Demo:** [https://piedy-pranav.github.io/ucu-growth-dashboard](https://piedy-pranav.github.io/ucu-growth-dashboard)

## Data Sources

| Source | Description | Period |
|--------|-------------|--------|
| [IPEDS](https://nces.ed.gov/ipeds) (NCES) | Fall enrollment & employee counts for all UCU partner universities | 2019–2024 |
| [NCUA](https://ncua.gov) 5300 Call Reports | UCU quarterly financials — membership, assets, loans, net worth | Q4 2019–2024 |
| [U.S. Census Bureau](https://data.census.gov) | Median household income by campus metro area | ACS 5-Year |

## Features

- **Opportunity Scorecard** — Composite scoring model ranking 14 partner schools by market size, growth momentum, financial need, and partnership freshness
- **Enrollment Trends** — Multi-year trend analysis with YoY change visualization
- **UCU Financials** — NCUA-sourced membership, asset, and loan growth with penetration analysis
- **School Detail** — Per-school deep dive with score breakdown and enrollment history

## Updating Data

1. Run the Jupyter notebook (`ucu_growth_analysis_v3.ipynb`)
2. The notebook exports JSON files directly to `dashboard/public/data/`
3. Refresh the browser (or redeploy)


## Methodology

The opportunity score is a weighted composite of four normalized (0–100) components:

- **Market Size (40%)** — Total addressable population (students + staff)
- **Growth Momentum (20%)** — Year-over-year enrollment change
- **Financial Need (20%)** — Inverse of metro area median household income
- **Partnership Freshness (20%)** — Inverse of partnership age (newer = more untapped)
